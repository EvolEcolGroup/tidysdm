# Coordinates of presences for lacertidae in the Iberian peninsula

Coordinates for presences of lacertidae, used as background for the
[`lacerta`](https://evolecolgroup.github.io/tidysdm/reference/lacerta.md)
dataset.. The variables are as follows:

## Usage

``` r
lacertidae_background
```

## Format

An `tibble` with 1,297 rows and 3 variables:

- ID:

  ids from GBIF

- latitude:

  latitudes in degrees

- longitude:

  longitudes in degrees
